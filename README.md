** Bagels – Number Guessing Game **

A simple command-line implementation of the classic **Bagels** number guessing game written in Python.

---

**Game Description**

The computer thinks of a **random 3-digit number**, and the player has **10 attempts** to guess it.

After each guess, the program provides clues:

- **Fermi** → One digit is correct and in the correct position.
- **Pico** → One digit is correct but in the wrong position.
- **Bagels** → No digits are correct.

The goal is to guess the exact 3-digit number within 10 tries.

---

**How to Run**

1. Make sure you have **Python 3** installed.
2. Clone or download this repository.
3. Navigate to the project directory in your terminal.
4. Run:

```bash
python bagels.py
